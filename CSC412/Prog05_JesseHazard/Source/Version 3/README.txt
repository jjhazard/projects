This is version 3 of the program to read single lines of code from multiple text files and write them to a single file. This program uses subprocesses to split the work into equal portions while executing new processes to save memory space. To compile the program, use the following command:

g++ version3.cpp

To compile the distributor executable, use:

g++ start3_distributor -o start3_distributor

To compile the processor exectutable, use:

g++ start3_processor -o start3_processor
